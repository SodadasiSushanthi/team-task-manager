# Team Task Manager

Full Stack Team Task Manager Web Application with Authentication, Role-Based Access, Project Management, Task Assignment, and Progress Tracking.

## Features

- User Authentication (Register/Login)
- JWT Token Authorization
- Role-Based Access Control (Admin / Member)
- Create and Manage Projects
- Assign Tasks to Team Members
- Task Status Tracking
- Dashboard Statistics
- Due Date & Overdue Task Tracking
- Protected Routes
- Responsive UI

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt.js
